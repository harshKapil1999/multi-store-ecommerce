# Admin Dashboard

Multi-tenant admin panel for managing stores, products, and orders built with Next.js 16 and React 19.

## Features

- Multi-store management
- Product inventory
- Order tracking
- User management
- Analytics dashboard
- Role-based access control

## Development

```bash
pnpm dev
```

Runs on http://localhost:3001

## Build

```bash
pnpm build
```

## Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:4000/api
```
