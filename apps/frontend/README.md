# Frontend App

Customer-facing e-commerce store built with Next.js 16 and React 19.

## Features

- Server-side rendering for SEO
- Product browsing and search
- Shopping cart
- Checkout flow
- Order history
- TanStack Query for data fetching
- Optimistic UI updates

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:4000/api
```
